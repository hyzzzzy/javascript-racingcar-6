# 🏎️ 2주차 자동차 경주

## 1. 게임 로직

### 1.1 게임 시작

- 경주할 N대의 자동차 이름 입력
- 이름 유효성 검사
- 시도할 횟수 입력
- 횟수 유효성 검사

### 1.2. 게임 진행

- 1대마다 랜덤값을 부여
- 랜덤값이 4이상이면 전진
- N대의 결과를 출력
- 시도할 횟수만큼 반복

### 1.3. 게임 종료

- 가장 많이 전진한 자동차 계산
- 우승자 출력

### 1.4. 폴더 구조

```
src
	- App.js
	- constants.js
    - Car.js
	- CarRace.js
	- index.js
	- utils.js
```

## 2. 구현 기능 목록

- 입력 함수
- 출력 함수
- 입력한 이름 유효성 검사 함수
    - 쉼표(,)를 기준으로 2개 이상의 이름이 작성되었는지?
    - 각 이름이 1이상 5이하의 글자 수를 가졌는지?
    - 중복된 이름이 없는지?
- 입력한 횟수 유효성 검사 함수
    - 숫자로 이루어졌는지?
    - 1 이상의 숫자를 입력했는지?
- 랜덤값 부여 함수
    - 0 ~ 9 사이의 랜덤값 지정
- 랜덤값 판별 함수
    - 0 ~ 9 사이의 랜덤값 중 4이상이 나왔는지 판별
- 판정 함수
    - 각 자동차마다 나온 랜덤값에 따라 판정
    - 4이상의 랜덤값이라면 전진횟수 증가
- 출력할 판정 결과 함수
    - `${이름} : -` 형식으로 출력
    - `-` 는 전진한 횟수만큼 출력
    - 마지막 공백 라인 추가
- 우승자 계산 함수
    - 가장 많이 전진한 자동차 이름 계산
- 출력할 우승자 결과 함수
    - 한 명일 경우, `최종 우승자 : ${이름}` 형식으로 출력
    - 여러 명일 경우,  `최종 우승자 : ${이름}, ${이름}` 형식으로 출력
